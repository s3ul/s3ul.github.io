## Problem
Typing arrows in plain text
## Solution
1. press `ctrl`+`shift`+`u`
2. type one of the 4-digit hexadecimal codes

| Character | Hex  |
| --------- | ---- |
| ←         | 2190 |
| ↑         | 2191 |
| →         | 2192 |
| ↓         | 2193 |
| ↔         | 2194 |
| ↕         | 2195 |
| ↖         | 2196 |
| ↗         | 2197 |
| ↘         | 2198 |
| ↙         | 2199 |
