## Problem
How to type arrow characters in plain text
## Solution
1. Press `Ctrl`+`Shift`+`U` (you should see an underlined 'u')
2. Type one of the 4-digit hexadecimal codes below
3. Press `Enter` or `Space` to insert the character

| Character | Hex  |
| --------- | ---- |
| ←         | 2190 |
| ↑         | 2191 |
| →         | 2192 |
| ↓         | 2193 |
| ↔         | 2194 |
| ↕         | 2195 |
| ↖         | 2196 |
| ↗         | 2197 |
| ↘         | 2198 |
| ↙         | 2199 |
